┌────────────────────────────────────────────────────────────────────────────┐
│ KEY IMPROVEMENTS OVER OLD CODE                                              │
├────────────────────────────────────────────────────────────────────────────┤
│ 1. STREAMING ARCHITECTURE                                                  │
│    • Old: Load entire 5GB file into memory                                 │
│    • New: Process line-by-line, use ~100MB RAM                            │
│                                                                             │
│ 2. SIMPLIFIED API                                                          │
│    • Old: corpus.build_corpus() → corpus.get_section_corpus()            │
│    • New: fetcher.fetch(categories=Category.MATH)                         │
│                                                                             │
│ 3. AUTOMATIC CACHING                                                       │
│    • Old: Manual pickle file management                                    │
│    • New: Transparent cache with automatic expiry                         │
│                                                                             │
│ 4. BUILT-IN FILTERS                                                        │
│    • Old: Custom selection functions for each query                        │
│    • New: Category enum + composable filters                              │
│                                                                             │
│ 5. PIP INSTALLABLE                                                         │
│    • Old: Copy-paste module files                                          │
│    • New: Standard pip install with proper dependencies                   │
└────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│ QUICK START                                                                 │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ 1. Install:                                                                │
│    pip install -e .                                                        │
│                                                                             │
│ 2. Download metadata:                                                      │
│    https://www.kaggle.com/datasets/Cornell-University/arxiv               │
│    → Save to ~/.arxiv_cache/arxiv_metadata.jsonl                          │
│                                                                             │
│ 3. Use:                                                                    │
│    from arxiv_metadata import ArxivMetadata, Category                     │
│    fetcher = ArxivMetadata()                                              │
│    df = fetcher.fetch(categories=Category.MATH, years=2024)              │

┌────────────────────────────────────────────────────────────────────────────┐
│ USAGE EXAMPLES                                                              │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ # All math papers from 2024                                                │
│ df = fetcher.fetch(categories=Category.MATH, years=2024)                  │
│                                                                             │
│ # Specific subcategories                                                   │
│ df = fetcher.fetch(categories=["math.AG", "math.NT"], years=2023)        │
│                                                                             │
│ # Multi-year range                                                         │
│ df = fetcher.fetch(categories=Category.CS, years=range(2020, 2025))      │
│                                                                             │
│ # Filter by author count                                                   │
│ df = fetcher.fetch(categories=Category.MATH, min_authors=2)              │
│                                                                             │
│ # Custom filter function                                                   │
│ df = fetcher.fetch(                                                        │
│     categories=Category.MATH,                                             │
│     filter_fn=lambda p: 'topology' in p['abstract'].lower()              │
│ )                                                                          │
│                                                                             │
│ # Memory-efficient streaming                                               │
│ for paper in fetcher.stream(categories=Category.MATH, years=2024):       │
│     print(paper['title'])                                                 │
└────────────────────────────────────────────────────────────────────────────┘
